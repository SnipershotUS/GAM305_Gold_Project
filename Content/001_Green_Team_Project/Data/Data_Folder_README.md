# Data Directory

## Purpose
Data tables, gameplay tags, and tunable values for designers.

## Data Tables (Ref #61)
Required tables:
- DT_Items: Item stats and properties
- DT_EnemyStats: Enemy health, damage, behavior
- DT_LootTables: Drop rates and rewards
- DT_TrapConfigs: Trap timing and damage
- DT_Objectives: Quest text and triggers

## Gameplay Tags (Ref #62)
Categories:
- Character states
- Damage types
- Item categories
- Quest states

## Tuning Variables (Ref #107)
- Expose in data assets
- Designer-friendly ranges
- No code changes required
