# Player Character Assets

## Modular Mesh System (Ref #32-36)

### Customization Slots
- Chest Armor (Common, Rare, Legendary)
- Weapon (3 Sword variants)
- Shield (Round, Kite)
- Helm (Optional)
- Cloak (Optional)

### Modular Meshes List (Ref #33)
- Base body
- Head
- Hair variants
- Chest armor variants
- Weapon variants
- Shield variants
- Cloak (optional)

### UV Layout Rules (Ref #34)
- Swappable parts must share texel density
- Consistent UV space allocation

### Texture Sheets (Ref #35)
- Body: One texture sheet
- Gear: Combined where possible

### Icon Requirements (Ref #36)
- 64x64 pixels for inventory
- 128x128 pixels for detailed view
- Simple silhouettes
- Transparent PNG format

## Animations List (Ref #37)
- Locomotion: Idle, Walk, Run, Turn
- Combat: Attack (light/heavy), Block
- Reactions: Hit React, Death
- Interactions: Pickup, Equip
